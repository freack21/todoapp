# TodoApp

TodoApp adalah web aplikasi manajemen tugas sederhana yang membantu Anda melacak dan mengatur tugas-tugas harian Anda. Web aplikasi ini dibuat menggunakan [VueJS](https://vuejs.org/) (via CDN).

## Fitur

- Tambah tugas baru
- Tandai tugas sebagai selesai
- Hapus tugas

## Deployment

Gunakan aplikasi ini di [deployment](https://freack21.github.io) !
